# Tailwind EDteam

A tailwind plugin that enables all styles guides of EDteam

## Installation

```sh
npm i tailwind-edteam
# or
yarn add tailwind-edteam
```

## Usage

In your `tailwind.config.js`

```js
module.exports = {
  content: [
    /* ... */
  ],
  theme: {
    extend: {
      /* ... */
    }
  },
  plugins: [require('tailwind-edteam')]
};
```
